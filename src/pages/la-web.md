# Sobre este sitio

Este sitio web está diseñado con simplicidad, rendimiento y sostenibilidad en
mente.

## Tecnología

- **Framework**: [Astro] - Sitio estático ultrarrápido
- **Estilos**: [Tailwind CSS] v4
- **Alojamiento**: [GitHub Pages] - hosting estático de carbono neutral
- **Lenguaje**: TypeScript para mayor seguridad de tipos
- **Tipografías**: Geist y Geist Mono descargadas desde [Fontsource]

## Sostenibilidad web

Este sitio sigue los principios de diseño web sostenible:

- HTML semántico y accesible
- CSS mínimo y optimizado
- Sin JavaScript innecesario en el cliente
- Hosting en servidores de bajo consumo
- Sin trackers ni scripts de terceros

### Huella de carbono

Puedes comprobar la huella de carbono de esta página en: [Website Carbon]

## Estándares web

Este sitio cumple con:

- HTML5 válido
- CSS3 moderno
- Accesibilidad WCAG 2.1
- Diseño responsive
- Progressive enhancement
